Matatalab Tale-Bot Pro

Tale-Bot Pro introduce pensamiento computacional a niños pequeños con un enfoque “screen-free”: programan el robot con botones físicos y señales luminosas, y pueden grabar voz para narrar historias.

Destacado​

  • Sin pantallas: comandos con botones y guías visuales de colores.
  • Creatividad: accesorios, dibujo y movimientos en secuencia.
  • Materiales seguros y resistentes para el aula.
  • En clase
  • Especificaciones
  • Recursos útiles
  • Grupos: 2 niños por robot.
  • Tiempo: 20–40 minutos.
  • Dinámica:
    1. Define una ruta con tarjetas o tapete.
    2. Introduce comandos (avanzar, girar, repetir).
    3. Graba una línea de narración y reproduce.
    4. Extiende con accesorios (dibujar, bailar).
  • Indicadores de color coherentes con los botones para facilitar el seguimiento.
  • Accesorios para dibujo y baile; tapetes/elementos para historias.
  • Carga y robustez pensadas para uso escolar.
  • Manuales y app/firmware: Centro de soporte
  • Currículo/actividades: Curriculum y actividades
  • Descarga: Hoja de inventario (CSV)
