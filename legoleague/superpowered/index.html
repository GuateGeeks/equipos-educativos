Instrucciones de construcción — Mission Models​

Instrucciones para ensamblar los modelos de misión incluidos en el set SUPERPOWERED (set 45820).

Cómo usar:

  1. Clasifica las bolsas de piezas LEGO por número (están etiquetadas).
  2. Busca el número de bolsa en la tabla y abre el PDF correspondiente.
  3. Ensambla un set a la vez para no mezclar piezas.

¿Faltan piezas LEGO? Visita el servicio de piezas faltantes de LEGO o llama al 1-800-422-5346 (EE.UU./CAN) · 00800 5346 5555 (Internacional). Menciona FIRST LEGO League.
