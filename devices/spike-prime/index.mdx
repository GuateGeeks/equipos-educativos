# LEGO SPIKE Prime

<div style={{display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '2rem', alignItems: 'start', marginBottom: '2rem'}}>
  <div>
      LEGO SPIKE Prime es un kit educativo para secundaria que combina construcción mecánica con programación visual tipo Scratch y Python. Está diseñado para apoyar clases de ciencias, tecnología, ingeniería y matemáticas con actividades prácticas y proyectos iterativos.

    <h2>Por qué elegirlo</h2>
    <ul>
      <li>Aprendizaje activo: construcción + codificación en un mismo entorno.</li>
      <li>Programación visual y Python: transiciones suaves según el nivel del grupo.</li>
      <li>Sensores y motores de calidad: respuestas fiables en el aula.</li>
      <li>Recursos curriculares y planes de clase: reduce la preparación del docente.</li>
    </ul>

<Tabs>
    <TabItem value="clase" label="En clase">
        - Grupos: 2–3 estudiantes por kit.
        - Tiempo típico: 45–90 minutos.
        - Flujo sugerido:
            1. Desafío breve y criterios de éxito.
            2. Construcción guiada con puntos de control.
            3. Programación incremental (bloques → variables → sensores).
            4. Pruebas, medición y presentación de resultados.
        - Evaluación: rúbrica simple (funciona, claridad del código, iteraciones).
    </TabItem>
    <TabItem value="especificaciones" label="Especificaciones">
        | Componente | Detalle |
        |------------|---------|
        | Hub | Pantalla 5×5 LED, 6 puertos, batería recargable |
        | Motores | Motores con encoder |
        | Sensores | Color/luz, distancia y movimiento (IMU en el hub) |
        | Conectividad | USB-C, Bluetooth |
        | Software | App LEGO Education (Scratch) y soporte Python |
    </TabItem>
</Tabs>

## Recomendaciones de uso

- Define metas claras por sesión (p. ej., "detectar color y reaccionar").
- Promueve roles: constructor/a, programador/a, tester; rota cada clase.
- Documenta con fotos/capturas y un breve diario de ingeniería.
