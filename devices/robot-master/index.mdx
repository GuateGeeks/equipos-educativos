# Makerzoid Robot Master (Premium)

Robot Master combina un amplio set de bloques con motores/servos y sensores para crear distintos robots móviles y estructurales. La app ofrece control remoto y programación por bloques para actividades graduales.

## Ventajas

- Variedad de modelos: fomenta creatividad y reutilización del set.
- Programación visual en la app y control inalámbrico.
- Enfoque STEAM: mecánica, electrónica básica y algoritmos.
