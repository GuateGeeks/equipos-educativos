# Tutoriales · Robot Master

Guías paso a paso para construir y programar robots.

## Primeros pasos

<TutorialStep step={1} title="Explora el kit">
Identifica bloques estructurales, motores, sensores y el módulo de control. Agrúpalos por tipo.
</TutorialStep>

<TutorialStep step={2} title="Instala la app">
Descarga la app Makerzoid, conecta el módulo vía Bluetooth y verifica la conexión.
</TutorialStep>

<TutorialStep step={3} title="Construye el modelo base">
Sigue las instrucciones del modelo "Carro básico" incluido en la app o manual.
</TutorialStep>

<TutorialStep step={4} title="Programa y prueba">
Arrastra bloques de movimiento, ejecuta y observa. Ajusta velocidad y duración.
</TutorialStep>

## Robot evasor de obstáculos

<TutorialStep step={1} title="Materiales">
Módulo de control, 2 motores, sensor de distancia/IR, bloques estructurales.
</TutorialStep>

<TutorialStep step={2} title="Construcción">
Monta la base con motores y ruedas; coloca el sensor apuntando al frente.
</TutorialStep>

<TutorialStep step={3} title="Lógica de programación">
Bucle: si distancia < 20 cm, detente, gira y continúa; de lo contrario, avanza.
</TutorialStep>

<TutorialStep step={4} title="Prueba y mejora">
Coloca obstáculos y observa el comportamiento. Ajusta umbrales y velocidad.
</TutorialStep>

:::info Extensión
Añade un LED que parpadee al detectar obstáculo.
:::
