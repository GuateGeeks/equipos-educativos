# Instrucciones de construcción · SUPERPOWERED℠

## Mission Models — Set 45820

Instrucciones para ensamblar los modelos de misión incluidos en el Challenge Set SUPERPOWERED.

**Cómo usar:**
1. Clasifica las bolsas de piezas LEGO por número (están etiquetadas dentro de la caja).
2. Localiza el número de bolsa en la tabla y abre el PDF correspondiente.
3. Ensambla un set a la vez para no mezclar piezas.
