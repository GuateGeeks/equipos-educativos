# BBC micro:bit

El micro:bit es una placa con sensores y LEDs que permite aprender código y electrónica de forma inmediata. Se programa por bloques (MakeCode) y Python, y se alimenta por USB o batería.

## Lo más útil

- Inicio rápido: display de LEDs, botones A/B y sensores integrados.
- Entorno versátil: bloques para principiantes y Python para avanzar.
- Amplio ecosistema: accesorios y proyectos para distintos niveles.

<Tabs>
  <TabItem value="clase" label="En clase">
    - Grupos: 2 estudiantes por placa.
    - Tiempo: 30–60 minutos.
    - Flujo:
      1. Prueba de entradas: botones y sensor de luz.
      2. Salidas: animaciones en LEDs y zumbador externo.
      3. Añade lógica: variables, condicionales y bucles.
  </TabItem>
  <TabItem value="especificaciones" label="Especificaciones">
    - LEDs en matriz 5x5.
    - Botones A/B, sensor de luz/temperatura, acelerómetro y brújula.
    - Comunicación: radio/BLE según versión.
    - Programación: MakeCode, MicroPython.
  </TabItem>

## Especificaciones técnicas

| Característica | Detalle |
|----------------|---------|
| LEDs | Matriz 5×5 |
| Botones | A, B y táctil (logo en v2) |
| Sensores | Luz, temperatura, acelerómetro, brújula |
| Conectividad | USB, radio 2.4 GHz, Bluetooth (v2) |
| Alimentación | USB o 2×AAA |

:::note Versiones
El micro:bit v2 añade micrófono, altavoz y más memoria.
:::
